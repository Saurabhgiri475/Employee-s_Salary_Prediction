# 💼 Employee Salary Prediction using Machine Learning

Predict employee salary categories using Machine Learning based on professional and demographic information. This project features a trained ML model with an interactive Streamlit web application for real-time salary prediction.

---

## 🚀 Live Demo

🔗https://employee-ssalaryprediction-123.streamlit.app/

---

## 📌 Project Overview

Employee Salary Prediction is a supervised Machine Learning project that predicts an employee's salary category based on various input features such as:

- Age
- Gender
- Education
- Experience Years
- Department
- Job Level
- Performance Rating
- Certifications
- Overtime Hours
- Remote Work
- City
- Company Tenure
- Projects Completed
- Skill Score

The project demonstrates the complete Machine Learning workflow including data preprocessing, model training, feature encoding, scaling, evaluation, and deployment using Streamlit.

## 📊 Dataset Features

|----------|-------------|

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Saurabhgiri475/Employee-s_Salary_Prediction.git
```

Move into the project folder

```bash
cd Employee-s_Salary_Prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run employee_salary_prediction.py
```
